GET     http://localhost:5000/users/login       Login in a account holder
POST	http://localhost:5000/users/register	Register a new user
GET	    http://localhost:5000/users	            Get all users
GET	    http://localhost:5000/users/:id	        Get user by ID
PUT	    http://localhost:5000/users/:id	        Update user by ID
DELETE	http://localhost:5000/users/:id	        Delete user by ID
